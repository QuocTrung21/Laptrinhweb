GIỚI THIỆU BẢN THÂN

Tôi là Nguyễn Quốc Trung, một lập trình viên đam mê khám phá và chia sẻ kiến thức về lập trình mạng. Tôi chuyên tìm hiểu về Java và JavaScript, với tập trung vào lập trình mạng và xây dựng các ứng dụng giao tiếp hiệu quả.

Thông qua blog này, tôi muốn chia sẻ những kiến thức tôi học được từ các khóa học về Java và JavaScript, đặc biệt là những bài học thực tế về lập trình mạng, Socket, HTTP Server, và giao tiếp Web. Tôi tin rằng chia sẻ kiến thức là cách tốt nhất để cùng nhau phát triển.

Kiến Thức Chính

  • Lập trình mạng với Socket, TCP/UDP trong Java
  • Xây dựng HTTP Server và Client ứng dụng
  • JavaScript Fetch API và WebSocket
  • Java Multithreading cho Network Server
  • Async/Await và xử lý bất đồng bộ trong JavaScript
  • REST API Client và Server Development
